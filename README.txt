Last run on Matlab R2017a.

1- Download these project files.

2- Go to https://www.rug.nl/ggdc/valuechain/wiod/wiod-2013-release to obtain WIOD data files in excel form.  Add these to the 'data' directory:

WIOT08_ROW_Sep12.xlsxWIOT00_ROW_Apr12.xlsxWIOT01_ROW_Apr12.xlsxWIOT02_ROW_Apr12.xlsxWIOT03_ROW_Apr12.xlsxWIOT04_ROW_Apr12.xlsxWIOT05_ROW_Apr12.xlsxWIOT06_ROW_Apr12.xlsxWIOT07_ROW_Apr12.xlsxWIOT09_ROW_Sep12.xlsxWIOT10_ROW_Sep12.xlsxWIOT11_ROW_Sep12.xlsxWIOT95_ROW_Apr12.xlsxWIOT96_ROW_Apr12.xlsxWIOT97_ROW_Apr12.xlsxWIOT98_ROW_Apr12.xlsxWIOT99_ROW_Apr12.xlsx

3- Open Matlab, navigate to the 'code' folder, and run main.m.

4- The data will be extracted from the excel files.  This will take several minutes.  When complete, you will be asked to move the processed data (saved to a .mat file) to a new location for further analyses.  Type 'y' and press enter to accept.  The program will continue, generating figures and other output.
